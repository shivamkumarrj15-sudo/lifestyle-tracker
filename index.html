Google Calendar API Setup Guide

बिना किसी फ़ाइल डाउनलोड के, सीधे अपने फोन कैलेंडर (Google Calendar) में रीयल-टाइम ऑटो-सिंक और अलार्म चालू करने की गाइड:

  1. Google Cloud Console खोलें: अपने वेब ब्राउज़र में Google Cloud Console खोलें और अपने Gmail अकाउंट से लॉगिन करें।
  2. नया प्रोजेक्ट बनाएं: सबसे ऊपर "Select a Project" पर क्लिक करें और "New Project" पर क्लिक करके प्रोजेक्ट का नाम (जैसे: Lifestyle Scheduler) रखें और Create दबाएं।
  3. Google Calendar API एक्टिव करें: सर्च बार में "Google Calendar API" लिखकर खोजें और उसे "Enable" करें।
  4. OAuth Consent Screen सेटअप करें: बाएँ साइडबार में "APIs & Services" > "OAuth Consent Screen" पर जाएँ। External चुनें और Create दबाएं। ऐप का नाम (जैसे: My Tracker) और अपनी ईमेल भरें। 'Save and Continue' करें।
  5. Test Users में ईमेल जोड़ें: Consent screen के "Test Users" सेक्शन में जाकर Add Users पर क्लिक करें और अपनी वही ईमेल आईडी जोड़ें जिससे आप सिंक करना चाहते हैं।
  6. Client ID बनाएं: "Credentials" टैब में जाएँ। ऊपर "+ Create Credentials" > "OAuth Client ID" पर क्लिक करें। Application Type में "Web Application" चुनें।
  7. Authorized Javascript Origins: Origins में http://localhost:8080 जोड़ें।
  8. कॉपी और पेस्ट करें: Client ID को कॉपी करें और डैशबोर्ड में बने इनपुट बॉक्स में पेस्ट करके Sync Google बटन पर क्लिक करें।
